# 雷思静个人作品集

这是一个可直接部署到 Vercel 或 GitHub Pages 的静态作品集网站。

## 内容

- 首页：蓝白色个人首屏、居中人物照片和 WELCOME 背景字
- 关于我：基础信息、教育背景、照片挂牌
- 我都自己尝试手搓了些什么？：单词贪吃蛇、单词地牢
- 实习经历：已预留后续补充位置

## 本地结构

- `index.html`：作品集首页
- `styles.css`：页面样式
- `app.js`：导航高亮交互
- `assets/`：个人照片、游戏图标、favicon
- `games/word-snake/`：单词贪吃蛇
- `games/word-dungeon/`：单词地牢
- `CNAME`：自定义域名配置

## 部署

把本文件夹内容推送到 `jiaozi-lsj/0706word-sanmingzhi` 仓库后，Vercel 会根据 GitHub 更新自动重新部署。
