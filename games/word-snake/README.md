# Word Snake MVP V1.0

纯前端静态网页版 2D 单词贪吃蛇游戏，按 `Word_Snake_MVP_V1.0_游戏规则与开发PRD.docx` 实现。

## 本地运行

```powershell
pnpm run dev
```

然后访问：

```text
http://localhost:5174/
```

## 构建

```powershell
pnpm run build
```

构建产物会生成在 `dist/`，可直接静态托管。

## 已实现范围

- Home / Game / Result 三屏与首次教程浮层。
- Canvas 2D 自动移动小蛇、路径跟随、半透明目标蛇节。
- 10 词测试词包、1 个正确中文能量 + 3 个干扰项。
- 吃对固化英文蛇节，吃错扣生命但不切换目标。
- 撞墙/撞身扣生命并短暂无敌，不计入单词错误。
- 180 秒倒计时、3 生命、完成/超时/生命耗尽结算。
- 结算展示完成数、首次正确数、错误次数、平均反应时间、薄弱词和最终蛇身。
- 键盘方向键/WASD、空格暂停、移动端四向触控、失焦自动暂停。
